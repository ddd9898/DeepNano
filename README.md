### Setup of DeepNano

The code is executed under `python=3.9` and  `torch=1.13.1+cu116`, we recommend you to use similar versions.

Install DeepNano: 

```shell
git clone https://github.com/ddd9898/DeepNano.git
cd DeepNano
pip install -r requirements.txt
```
